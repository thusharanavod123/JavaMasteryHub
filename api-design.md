# API Design Best Practices  
## 1. REST vs. GraphQL  
- **REST:** Standard HTTP methods (GET, POST, PUT, DELETE)  
- **GraphQL:** Fetch only required data  

## 2. Versioning  
- **URI versioning:** `/api/v1/users`  
- **Header versioning:** `Accept: application/vnd.company.v1+json`  

## 3. Authentication  
- **JWT (JSON Web Tokens)**  
- **OAuth 2.0**
